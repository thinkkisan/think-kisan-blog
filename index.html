Think Kisan: 7-in-1 Soil Monitoring & Smart Irrigation

Real-time soil telemetry + automated irrigation using LoRa, ESP32 & Firebase.

Think Kisan Team January 15, 2025

Project Overview

Think Kisan is a compact farm telemetry system using a 7-in-1 soil sensor connected to an Arduino. Data is forwarded over LoRa to an ESP32 gateway which pushes data to Firebase (Firestore + Realtime DB). An Arduino Nano with a 40A relay controls the irrigation motor based on field water level and soil moisture. We built a multi-user surveillance mobile app in Expo React Native that shows dashboards, operation controls (manual/auto), weather (via free weather API), and exports soil data to PDF by averaging last N values or within a time period.

Hardware Components

The Think Kisan system is built with carefully selected components for reliability and performance in agricultural environments.

Sensor & Data Collection

  • 7-in-1 Soil Sensor Measures moisture, temperature, pH, nitrogen, phosphorus, potassium, and electrical conductivity
  • Arduino (Sensor Box) Collects and processes sensor data before transmission
  • LoRa Modules Long-range wireless communication (up to 10km in open field)

Gateway & Control

  • ESP32 Gateway Receives LoRa data and connects to Firebase via WiFi
  • Arduino Nano Controls irrigation system based on sensor thresholds
  • 40A Relay + Motor High-capacity relay for irrigation pump control

Data Flow & Communication

Think Kisan uses a multi-layer communication architecture to ensure reliable data transmission from field to cloud.

Communication Architecture

The system employs a hybrid communication strategy combining LoRa for long-range field communication and WiFi for cloud connectivity. This approach ensures reliable data transmission even in remote agricultural areas where WiFi coverage may be limited.

1

Sensor → Arduino Box

7-in-1 soil sensor communicates with Arduino via RS485/Modbus protocol

2

Arduino Box → LoRa

Arduino packages data and transmits via LoRa module (433MHz/868MHz/915MHz)

3

LoRa → ESP32 Gateway

ESP32 receives LoRa packets and decodes sensor data

4

ESP32 → Firebase

Gateway pushes data to Firestore (historical) and Realtime Database (live status)

Cloud & Databases

Think Kisan leverages Firebase's dual-database architecture for optimal performance and flexibility.

Firestore

Purpose: Historical sensor data storage

  • Time-series soil measurements
  • Irrigation event logs
  • User activity records
  • Optimized for complex queries and analytics

Realtime Database

Purpose: Live system status monitoring

  • Circuit box charging status
  • Current water moisture levels
  • Temperature of 3 circuit boxes
  • Optimized for real-time synchronization

Irrigation Control System

Intelligent irrigation management with automatic and manual control modes.

Control Logic

The irrigation system operates in two modes, providing flexibility for different farming scenarios and user preferences.

Automatic Mode

Arduino Nano continuously monitors soil moisture and water level sensors. When moisture drops below threshold AND water level is sufficient, the 40A relay activates the irrigation motor automatically.

Threshold-based Energy efficient 24/7 monitoring

Manual Mode

Users can override automatic control via the mobile app. Direct ON/OFF commands are sent through Firebase to the Arduino Nano, providing immediate control over irrigation operations.

User override Instant control App-based

Mobile Application

Built with Expo React Native for cross-platform compatibility and real-time data visualization.

Dashboard

Real-time visualization of all sensor data with charts and graphs. Monitor soil conditions, irrigation status, and system health at a glance.

Operations Control

Switch between automatic and manual irrigation modes. Direct control over irrigation motor with real-time status feedback.

Weather Integration

Live weather data based on device location using free weather API. Helps plan irrigation schedules around rainfall predictions.

Document Export

Export soil data to PDF format. Choose to average last N values or filter by time period for comprehensive reports.
